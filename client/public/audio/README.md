# Audio Files for Adzan & Shalawat Tarhim

Letakkan file audio Anda di folder ini dengan nama berikut:

| File              | Fungsi                                       | Format |
| ----------------- | -------------------------------------------- | ------ |
| `adzan.mp3`       | Adzan standar (Dzuhur, Ashar, Maghrib, Isya) | MP3    |
| `adzan_subuh.mp3` | Adzan Subuh (opsional, jika berbeda)         | MP3    |
| `tarhim.mp3`      | Shalawat Tarhim untuk waktu Imsak            | MP3    |

## Catatan

- Jika `adzan_subuh.mp3` tidak ada, sistem akan menggunakan `adzan.mp3`
- Shalawat Tarhim akan diputar 15 menit sebelum waktu Imsak (default)
- Volume dapat diatur dari dashboard
